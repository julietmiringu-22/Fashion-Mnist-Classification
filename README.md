# Fashion-Mnist-Classification
This project implements Convolutional Neural Network (CNN) using keras in both R and python to classify and recognize images from a selection of 10 clothing items.
# Model
The CNN consists of 6 layers Maxpooling 2D, Conv2D, Flatten ,Dense and the output layer.

# User Instructions
Install Tensorflow or keras, Numpy and Matplotlib. These are the most important libraries needed to import the data and visualize it. 
# Contributor Expectations
Process multiple images at the same time

# Known Issues
The keras package in R creates a challenge while running the code.
